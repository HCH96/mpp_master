3. 앱 기능
<하루일정 추가 페이지>
- 하루 단위의 일정을 추가시킬 수 있음
- 시간대별(아침, 점심, 저녁)로 화면에 할일이 표시됨
- 할일을 저장할 때, 카테고리(미지정, 운동, 과제 등)을 지정할 수 있음

<하루일정 diagram 페이지>
- piechart를 통해 하루 일정을 보여줌

<memo 페이지>
- 날짜별로 메모를 저장할 수 있음

<월별 일정 추가 페이지>
- 월별 일정을 추가할 수 있음
- 시작 요일과 종료 요일을 지정할 수 있음
- 해당 일자에 요일이 있으면, calender view에 작은 점(red)이 표시된다.

<검색 페이지>
- 해당 키워드가 있는 일정을 검색할 수 있음
- 월별, 일별 일정을 모두 검색할 수 있음
